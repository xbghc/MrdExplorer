# MRD Explorer

## 使用方法

左上角的文件浏览框中`按文件`和`按文件夹`两个选项：

如果选择`按文件`，则会隐藏目录下的文件夹并只显示`.mrd`文件，
点击列表中的文件，就会在界面的右侧显示成像结果。

`按文件夹`通常用于整理后的数据，每一个文件夹中包含一次扫描中不同通道的数据，
点击文件夹，就会显示文件夹中所有文件的成像结果，每一列是一个通道（通道即文件）

## 依赖

- PySide6(Qt6.8)
- Python3.12
